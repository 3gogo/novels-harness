# 华语网络小说套路大全 (2024-2026)

## 一、男频 (Male-Oriented)

### 1. 玄幻类
| 套路 | 说明 |
|------|------|
| **废柴流** | 主角开局最弱/无天赋，发现金手指后逆袭。代表作《斗破苍穹》 |
| **退婚流** | 开篇被未婚妻当众退婚羞辱，立誓证明价值 |
| **升级流** | 明确的等级体系，从凡人到仙人/神的系统进阶 |
| **打脸流** | 反派嚣张→主角隐藏实力→当众打脸反转 |
| **装逼打脸** | 主角故意隐藏实力被嘲讽，然后亮出真实力 |

### 2. 修仙/仙侠类
| 套路 | 说明 |
|------|------|
| **凡人流** | 普通人无背景修炼到巅峰。代表作《凡人修仙传》 |
| **洪荒流** | 基于中国神话宇宙观（盘古、女娲、鸿钧等） |
| **无限流** | 穿梭不同"副本世界"完成任务获取力量。代表作《无限恐怖》 |
| **签到流** | 在特定地点"签到"获得奖励。2020年代热门 |
| **苟道流** | 主角隐藏实力、避事低调修炼。代表作《我真的是修炼大佬》 |
| **稳健流** | 极度谨慎+隐藏实力 |

### 3. 都市类
| 套路 | 说明 |
|------|------|
| **都市修仙** | 修炼者生活在现代社会 |
| **都市兵王** | 前特种兵精英回归平民生活 |
| **神豪流** | 突然获得无限财富，享受奢华生活 |
| **奶爸流** | 前战士变身全职奶爸 |
| **赘婿流** | 入赘豪门被看不起，暗中拥有强大能力。代表作《龙王赘婿》 |
| **直播流** | 主角成为主播，通过直播获取粉丝和力量 |

### 4. 科幻/末世类
| 套路 | 说明 |
|------|------|
| **末世流** | 丧尸爆发/核废土/资源匮乏生存 |
| **星际流** | 太空探索、银河帝国 |
| **机甲流** | 巨型机器人驾驶 |
| **进化变异** | 生物进化、超能力觉醒 |
| **末世囤货** | 预知末日来临，疯狂囤积物资 |

### 5. 历史类
| 套路 | 说明 |
|------|------|
| **穿越历史** | 现代人穿越到古代朝代 |
| **架空历史** | 虚构历史背景，借鉴真实朝代 |
| **种田流** | 从0开始建设领地：农业、贸易、军事 |

### 6. 游戏类
| 套路 | 说明 |
|------|------|
| **网游/VRMMO** | 虚拟现实游戏世界 |
| **电竞流** | 专业竞技比赛 |

## 二、女频 (Female-Oriented)

### 1. 言情类
| 套路 | 说明 |
|------|------|
| **甜宠** | 暖甜恋爱，男主宠溺女主，无虐点 |
| **虐恋** | 误会、背叛、被迫分离的情感过山车 |
| **先婚后爱** | 包办/契约婚姻转变为真爱 |
| **青梅竹马** | 从小一起长大的CP |
| **破镜重圆** | 因误会分手的前任复合 |
| **替身** | 女主是男主初恋的替代品 |
| **追妻火葬场** | 男主严重伤害女主后追悔莫及，疯狂挽回。近年大热 |
| **带球跑** | 女主怀孕出逃独自抚养，男主后来找到 |

### 3. 现代类
| 套路 | 说明 |
|------|------|

| 套路 | 说明 |

| **重生** | 女主死后重生回过去，利用未来知改变命运 |

## 三、通用套路设定 ("金手指"系统)

| 金手指类型 | 说明 |
|------------|------|
| **系统流** | 游戏化界面，任务/经验值/奖励 |
| **老爷爷** | 神器/戒指中住着强大导师灵魂。代表作药老《斗破》 |
| **血脉觉醒** | 发现隐性恐怖血脉 |
| **神器/法宝** | 获得只有自己能激活的超神装备 |
| **重生预知** | 知道未来并加以利用 |
| **抽奖/氪金** | 抽卡获取物品/技能/能力 |
| **签到系统** | 每日签到获取递增奖励 |

## 六、2024-2026 新兴/热门趋势

| 趋势 | 说明 |

| **群像文** | 关注多角色非单一MC |
| **无CP** | 刻意去掉恋爱线的女性向作品 |
| **末世囤货** | 预知末日囤物资 |

## 七、故事结构模式

| 结构 | 说明 |
|------|------|
| **长篇升级流** | 数千章持续升级，男频标准结构 |
| **快穿世界** | 单元剧结构，MC进入不同世界完成任务 |
| **系统任务** | 系统发布任务驱动叙事 |
| **无限副本** | 连续不断的恐怖/生存"副本" |
| **重生改命** | 利用未来知识_undo_悲剧命运 |
| **穿书反套路** | 穿越进小说但主动颠覆原剧情 |
| **马甲暴露** | 女主秘密身份逐一暴露（戏剧性讽刺） |
